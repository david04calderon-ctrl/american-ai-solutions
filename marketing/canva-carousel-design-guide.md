# Canva LinkedIn Carousel Design Guide
# American AI Solutions LLC

**Purpose:** Step-by-step instructions to build the 4 LinkedIn carousel PDFs. Each carousel takes 60–90 minutes the first time; 20–30 minutes once you have the template.
**Output:** PDF files, 1920×1080 px (LinkedIn carousel standard)
**Brand assets needed:** Colors from brand kit (below), Google Fonts (DM Sans + DM Serif Display)

---

## BRAND KIT — SET THIS UP FIRST IN CANVA

Go to **Canva → Brand Kit** (available on free plan).

### Brand Colors (add all of these)

| Name | Hex Code | Usage |
|---|---|---|
| Primary Blue | `#1A3A8F` | Headlines, borders, accent bars, buttons |
| Dark Background | `#0D1117` | Dark slides (cover, closing slides) |
| Demo Card | `#101825` | Dark card backgrounds |
| Demo Border | `#1E2D45` | Dark section borders |
| Cyan Glow | `#00D4FF` | Highlight text, stat numbers, accents |
| Gold Aura | `#FFD700` | Key data points, dollar signs |
| Mid Blue-Gray | `#4A5473` | Body text on light slides |
| Light Gray BG | `#F7F9FC` | Section alt background |
| Border Gray | `#E4E8F0` | Light section borders |
| Pure White | `#FFFFFF` | Text on dark slides, light slide backgrounds |

### Brand Fonts

In Canva, you may need to search for these in the font selector:

**Headings:** DM Serif Display (italic available)
**Body text:** DM Sans

## CANVA SETUP — CREATE YOUR TEMPLATE

### Step 1: Create a new design
1. Canva.com → Create a design → Custom size
2. Width: **1920 px**, Height: **1080 px**
3. Name it: "LinkedIn Carousel Template"

### Step 2: Build the reusable slide elements

**The vertical accent bar** (appears on all slides)
- Rectangle → 16px wide × 120px tall
- Fill: `#1A3A8F`
- Position: Left edge, vertically centered

**The company tag** (bottom-left of every slide)
- Text: "American AI Solutions LLC | Detroit, MI"
- Font: DM Sans, 22px, weight 500, color `#4A5473`
- Position: Bottom-left, 48px from left edge, 32px from bottom

**Slide number** (bottom-right of every slide)
- Text: "1/9" (change per slide)
- Font: DM Sans, 22px, weight 500, color `#4A5473`
- Position: Bottom-right, 48px from right edge, 32px from bottom

## SLIDE LAYOUTS

### Layout A: Dark Cover Slide (Slide 1 and Slide 9)

Background: `#0D1117`

Elements:
- Left accent bar: 12px × 180px, color `#1A3A8F`, left edge vertically centered
- Tag text: All-caps label (e.g., "MICHIGAN MANUFACTURING") — DM Sans, 18px, weight 700, letter-spacing 0.1em, color `#00D4FF`
- Main headline: DM Serif Display, italic, 64–72px, white, 2–3 lines max
- Subheadline (optional): DM Sans, 26px, weight 300, color `#9AA5B8`, below main headline
- Bottom-left: "American AI Solutions LLC | americanaisolutionsllc.com" — DM Sans, 22px, color `#4A5473`

**Canva tip:** Use a solid background rectangle (#0D1117) as the base layer — don't use Canva's background fill directly, as it doesn't export cleanly in PDF mode.

---

### Layout B: White Content Slide (Slides 2–7)

Background: White `#FFFFFF`

Elements:
- **Slide number bar at top:** 1920px × 8px rectangle, color `#1A3A8F`, pinned to top
- **Slide label** (top-left below bar): All-caps tag — DM Sans, 16px, weight 700, letter-spacing 0.1em, color `#4A5473`
- **Main point headline:** DM Serif Display, 48–56px, color `#0D1117`, line-height 1.2
- **Body text** (if needed): DM Sans, 26–28px, weight 400, color `#4A5473`, line-height 1.5
- **Highlight stat** (if a number is the focus): DM Sans, 64–80px, weight 700, color `#1A3A8F` or `#00D4FF`
- **Bottom rule:** 1px line, color `#E4E8F0`, 40px from bottom
- **Company tag + page number:** DM Sans, 22px, color `#4A5473`

---

### Layout C: Comparison or List Slide

Background: `#F7F9FC` (light gray)

Elements:
- Same top bar and header as Layout B
- For lists: use a vertical left border line (`#1A3A8F`, 3px wide) with list items to the right
- For comparisons: two-column layout, left column = "Before" (gray tones), right column = "After" (blue/cyan tones)
- Use a `#00D4FF` highlight color for the winning number or "After" state

---

### Layout D: Stat/ROI Slide

Background: `#0D1117` (dark, but less stark than the cover)

Elements:
- Large stat in center: DM Sans, 96–120px, weight 700, color `#00D4FF`
- Stat label above: DM Sans, 24px, weight 500, color `#9AA5B8`
- Supporting text below: DM Sans, 28px, color `#9AA5B8`, max 2 lines
- Small left accent bar, `#1A3A8F`

## EXPORTING THE CAROUSEL PDF

1. In Canva → Share (top right)
2. Download → **PDF Standard** (not PDF Print)
3. Check: "All pages" selected
4. Download

**File naming:** `AAIS-Carousel-01-Automotive-[Month]-[Year].pdf`

Upload to LinkedIn as a **Document** post (not an image post). LinkedIn renders the PDF as a swipeable carousel automatically.

## CANVA SHORTCUTS AND TIPS

**Duplicating slides:** Click a slide in the panel → `Ctrl+D` (Windows) or `Cmd+D` (Mac) → adjust text

**Aligning elements:** Select multiple elements → click the "Position" button in the top toolbar → Align options

**Consistent spacing:** Select elements → right-click → "Space evenly" (horizontal or vertical)

**Color consistency:** Add brand colors to your palette first (Brand Kit) → click any element → color selector → your saved colors appear at the top

**Text size for readability on mobile:** LinkedIn carousels are viewed on phones. Test by zooming your browser to 25% — if you can still read the headline, it's large enough. Body text should be 26px minimum.

**Save as a template:** Once you've built slide 1 perfectly, duplicate the entire design → rename as "AAIS Carousel Template" → use this as the starting point for every new carousel.
