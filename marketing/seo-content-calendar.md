## SEO TECHNICAL REQUIREMENTS

For each post, ensure:
- **Title tag:** Under 60 characters, includes primary keyword
- **Meta description:** 150–160 characters, includes keyword, has a hook
- **H1:** Matches or closely mirrors the title tag
- **H2/H3 headings:** Include secondary keywords naturally
- **Internal links:** Each post links to 2–3 other posts AND the contact/pricing page
- **External links:** Link to 1–2 authoritative sources (MEDC, BLS, MDOT) — signals credibility
- **Image alt text:** Describe the image with keyword context
- **URL slug:** Short, keyword-rich (/ai-quality-inspection-michigan-manufacturer/, not /post-1/)
- **Schema markup:** Article schema with author (David Calderon), datePublished, dateModified

## HOW TO ADD A BLOG TO THE SITE

**Option A — Static files in Netlify (free, no CMS needed initially):**
- Create a `/blog/` directory in the repo
- Write each post as an HTML file with the same CSS as the main site
- Link to `/blog/` from the navbar
- Netlify serves them automatically

**Option B — Netlify + Decap CMS (free, adds visual editor):**
- Decap CMS (formerly Netlify CMS) connects to your GitHub repo
- You get a visual editor at `/admin` — write posts without touching HTML
- Posts save as markdown files, Netlify builds them
- Setup: ~2 hours

**Option C — Ghost (if you want a full blog platform):**
- Ghost Pro starts at $9/month — professional publishing platform
- Can embed in a subdomain: `blog.americanaisolutionsllc.com`
- Better for SEO long-term (proper blog architecture)
- Start here once you're posting regularly

**Recommendation:** Start with Option A (static HTML) for the first 2–3 posts. Move to Decap CMS after 3 posts if you want to keep writing without touching code.
